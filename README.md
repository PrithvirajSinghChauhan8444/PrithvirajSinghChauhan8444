### 🚀 Core Projects (AI & Web)

#### 🤖 [Personal AI Assistant](https://github.com/PrithvirajSinghChauhan8444)

* **Status**: ![Status](https://img.shields.io/badge/Status-In%20Development-yellow?style=flat-square) `[Flagship Project]`
* **Description**: A secure, local-first personal assistant built on a modular, **Dynamic State-Graph Multi-Agent Architecture** (LangGraph). It decomposes complex user requests into structured, sequential subtasks and delegates them to sandboxed worker agents.
* **Key Architecture & Tech**:
  * **Orchestration Engine**: Cyclic state machine utilizing LangGraph to coordinate workers, prevent context pollution/drift, and resolve execution deadlocks.
  * **Sandboxed Worker Agents**: Independent ReAct workers (System, Gmail, Productivity, GitHub, and Google Classroom) equipped with target-specific python tools.
  * **Nested Obsidian Team**: A specialized sub-graph team featuring an Obsidian Manager and sub-workers (Note, Canvas, and Refactor agents) with dynamic folder nesting.
  * **Zero-Trust Security**: High-risk OS and shell commands are isolated behind a password verification layer and standard input mutex.
  * **Context & Performance**: Semantic vector search via FAISS (`all-MiniLM-L6-v2`) for skill discovery, and ChromaDB for user preference learning.
  * `Python` `LangGraph` `LangChain` `Gemini` `FAISS`

#### 🔗 [HermesLink](https://github.com/PrithvirajSinghChauhan8444/HermesLink)

* **Status**: ![Status](https://img.shields.io/badge/Status-Completed-brightgreen?style=flat-square)
* **Description**: A distributed, real-time remote download manager that separates download intent (submitted via web dashboard) from physical execution (run by Python agents on target devices).
* **Key Architecture & Tech**:
  * **Event Bridge Sync**: Combines Firestore (`on_snapshot` zero-polling dispatching) with Realtime Database (low-latency, sub-second progress updates and pause/resume/stop actions).
  * **Pluggable Download Engines**: Uses the strategy pattern to coordinate Aria2 (multithreaded JSON-RPC downloads), yt-dlp (dynamic video extraction format selector), and direct HTTP/HTTPS engines.
  * **Smart Categorization**: AI-powered classifier dynamically routes downloaded files into organized folders (Media, Apps, Archives) based on URL and filename properties.
  * **DevOps & Service Design**: Configured for background execution as daemon systemd user services.
  * `React` `Vite` `FastAPI` `Python` `Firebase RTDB + Firestore` `Aria2`
